📊 ANÁLISIS DE SENTIMIENTOS

El análisis de sentimientos permite identificar la polaridad emocional de un texto, clasificándolo como positivo, negativo o neutral mediante el conteo de palabras clave. 

# Definir listas de palabras emocionales
positivas = ["feliz"]
negativas = ["perdido", "frustrado", "triste", "insoportablemente", "confundido"]

# Texto a analizar
texto = ("feliz feliz perdido")

# Tokenización y normalización
tokens = texto.lower().split()
conteoposi = 0
conteonega = 0

# Análisis de sentimientos
for palabra in tokens:
    if palabra in positivas:
         conteoposi += 1
    elif palabra in negativas:
        conteonega  += 1

# Resultados
print("Palabras positivas encontradas:", conteoposi)
print("Palabras negativas encontradas:", conteonega)

# Clasificación final
if conteonega > conteoposi:
    print("La frase es negativa.")
elif conteoposi > conteonega:
    print("La frase es positiva.")

📚 Glosario de Funciones

lower()

La función lower() en Python se utiliza para convertir todo un texto o un token (palabra) a minúsculas. Esto es fundamental en el procesamiento de texto para normalizar el contenido y asegurar que las comparaciones sean insensibles a mayúsculas/minúsculas.

💡 Ejemplo de uso: texto.lower() → "hola mundo"

split()

La función split() en Python se usa para dividir una cadena de texto en partes (normalmente palabras), y es clave para generar tokens de manera sencilla. Por defecto, divide el texto usando espacios en blanco como separador, retornando una lista de tokens.

💡 Ejemplo de uso: "hola mundo".split() → ['hola', 'mundo']
