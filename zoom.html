📷 AMPLIACIÓN Y REDUCCIÓN DE IMÁGENES

📝 Código del Ejemplo

Programa interactivo para hacer zoom en imágenes usando teclas: '+' para ampliar, '-' para reducir, 'o' para restaurar tamaño original y 'x' para salir. 

def zoom():
    import cv2
    
    # Carga la imagen
    img_ori = cv2.imread("Spider.jpg")
    
    # Creando variable para el valor del zoom original
    zoom = 1.0

    while True:
        img = cv2.resize(img_ori, None, fx=zoom, fy=zoom)
        cv2.imshow("imagen nueva", img)
        tecla = cv2.waitKey(0)
        
        if tecla == ord('+'):
            zoom = zoom * 1.2
        elif tecla == ord('-'):
            zoom = zoom * 0.5
        elif tecla == ord('o'):
            zoom = 1.0
        elif tecla == ord('x'):
            break
    
    cv2.destroyAllWindows()

if __name__ == "__main__":
    zoom()

📚 Glosario de Funciones

zoom

Su función principal es ampliar o reducir (hacer zoom) de un objeto, imagen o arreglo. En este caso, es una variable que almacena el factor de escala para redimensionar la imagen.

None

Es un valor especial que representa la ausencia de un valor o un valor nulo. En cv2.resize(), se usa None cuando no especificas dimensiones fijas y prefieres usar factores de escala (fx, fy).

resize()

Se usa principalmente para cambiar el tamaño de estructuras de datos como listas, arrays (matrices) o imágenes, dependiendo de la librería que estés usando. En OpenCV, redimensiona imágenes según las dimensiones o factores especificados.
